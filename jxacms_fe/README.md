## git 分支管理
线上分支 master
开发分支 dev
开发分支命名采用小驼峰  功能-日期-姓名
开发完毕的分支，请合到dev, 测试dev完毕，上线分支合到master， 上线后请定期删除已开发完毕分支
每天需要dev 分支从 master 当中拉取代码
